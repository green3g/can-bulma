<file-field />

module
{can.Component}
form-widget/field-components/file-field  
  • Input Accept Types

A file upload field. Once a file is selected, this component POST's the upload to a special url and returns the url result as text to the form. This is experimental!

can.Component

Properties

Name Type Description

Whether or not to allow multiple uploads

The type of media to accept
