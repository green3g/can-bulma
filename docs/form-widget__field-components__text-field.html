<text-field />

module
{can.Component}
form-widget/field-components/text-field  
  • HTML Input Types

A generic text box with a label. If the textType property is not provided, 'text' is the default.

can.Component

Properties

Name Type Description
textType {String}

The type of textbox, this can be any html input type. The default is text
fieldType {String}

This is set to text by default and does not need to be provided
textarea {Boolean}

Set to true if the input textbox should be a large textarea
