module
{can.Component}
form-widget/field-components/checkbox-field  

A checkbox style field.

Example

{
  name: 'op',
  alias: 'is',
  fieldType: 'checkbox',
  style: 'switch' // the default is 'checkbox'
},

can.Component

Properties

Checkbox style, this can be checkbox or switch
