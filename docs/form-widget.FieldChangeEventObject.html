FieldChangeEventObject

typedef
{Object}
form-widget.FieldChangeEventObject  

This event object is dispatched whenever a field in a form changes

Object

Properties

Name Type Description
name {String}

The name of the field
value {Any}

The current value of the field, this can be any type
current {Object}

The current saved values that were passed to the form object performing the validation. This has not been updated yet from any form changes that occurred since the last submit
dirty {Object}

An object consisting of the values of the fields that have changed since the last submit.
