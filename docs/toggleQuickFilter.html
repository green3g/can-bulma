toggleQuickFilter

function
 

A helper for toggling quick filter dropdowns from the template, this accepts simplified parameters, similar to the addFilter function. Depending on the arguments several outcomes may occur:

function(fieldName, value, event)

Parameters

Name Type Description
fieldName {String}

The field name to filter on
value {Any}

The value to filter on
event {Event}

The dom click event to cancel

Returns


{Boolean}

Returns false to prevent page navigation

  • If a filter does not exist with the given field name, it will be created
  • If a filter does exist with the field name, but the value is different, the filter will be updated with the new value
  • If the filter does exist with the current value, the filter will be removed The operator used will be 'equals'
