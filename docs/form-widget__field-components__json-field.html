<json-field />

module
{can.Component}
form-widget/field-components/json-field  

DEPRECATED. Use subform-field instead. This component creates a sub form that is serialized to a JSON string when values in the sub form change. This can be used to store JSON data in a database field.

Example

{
  ObjectTemplate: DefineMap.extend({
    //object properties
    field1: 'string',
    field2: 'number'
  }),
  fieldType: 'json'
}

can.Component

Properties

Name Type Description
fieldType {String}

This should be set to json for the form widget to use this template
ObjectTemplate {can.Map}

The object to use as the form object template
formFields {Array<String | Field>}

The form fields to use in the form
