<select-field />

module
{can.Component}
form-widget/field-components/select-field  

A select dropdown with options. See selectFieldProperty and selectOptionProperty.

Example

{
name: 'op',
alias: 'is',
placeholder: 'Choose a operator',
fieldType: 'select',
options: [{
 label: 'Equal to',
 value: '=='
}, {
 label: 'Not equal to',
 value: '!='
}, {
 label: 'Contains',
 value: 'in'
}, {
 label: 'Does not contain',
 value: 'not_in'
}, {
 label: 'Like',
 value: 'like'
}]
},

can.Component

Properties

Name Type Description
fieldType {String}

This should be set to select to use this template in the form widget
options {Array<spectre.types.SelectOptionProperty>}

An array of values and labels
optionsPromise {Promise}

A promise that asynchronously resolves to an array of SelectOption
